# ML_Methodlari_ile_Meme Kanseri Tahmini ve Aciklanabilirligi
Bu proje, Wisconsin Meme Kanseri veri setiyle malignite tahmini için uçtan uca bir ML çözümü sunar. Veri ön işleme, özellik mühendisliği, model karşılaştırması, optimizasyon ve SHAP ile açıklanabilirlik içerir. Yüksek doğruluklu, şeffaf bir model geliştirerek klinik karar süreçlerine destek olmayı hedefler.
